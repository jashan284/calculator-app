🧮 Calculator App

A simple and user-friendly calculator application built using Python and tkinter
🚀 Features
- Perform basic arithmetic operations (+, -, ×, ÷)
- Clean and easy-to-use interface
- Fast and lightweight application

🛠️ Technologies Used
- Python
- Tkinter (GUI library)

 ▶️ How to Run
1. Install Python on your system
2. Download or clone this repository
3. Open the file calculator.py
4. Run the file using Python

🎯 Purpose
This project was created as a beginner-level practice to learn Python GUI development and understand how desktop applications are built.
